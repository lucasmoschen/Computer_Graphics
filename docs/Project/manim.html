Manim

É uma ferramenta de animação desenvolvida por Grant Sanderson, do canal 3Blue1Brown. A ferramenta faz com que as provas matemáticas fiquem mais simples de serem implementadas, na linguagem Python. É uma ótima ferramenta para obter resultados similares aos obtidos no cana. Ela lida com animações 2D e 3D.

Manim é formado por Python (a linguagem de programação), LaTex (lida com a parte textual), Cairo (programa que faz as figuras), FFmpeg (agrupa os frames para formar o vídeo) e SoX (programa de áudio). É um software livre e é customizável. A documentação ainda não é muito clara, mais muito interessante.

Cada classe representa uma animação. Criamos uma classe do tipo Cena para iniciar os trabalhos. Devemos criar o método contruct, que funciona como o init da animação. Para interpretar um código, usa-se "python -m manim name.py NameOfClass -[symbol]. Symbol pode ser p, que sifnifica preview, h, high quality ou l, low quality.
